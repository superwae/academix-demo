Additional Highlights

🎨 Theme Engine

18 built-in themes (14 solid + 4 gradient) plus full custom theme support via CSS variables and HSL color picker. Persisted with Zustand + localStorage.

📚 Course System

Full lifecycle: Draft → Published → Archived. Supports video lessons, materials, sections, multiple modalities (Online/In-Person/Hybrid), and auto-certificates on completion.

📝 Assessment Engine

Assignments with multi-attempt submissions & file uploads. Exams with MCQ, True/False questions, timed attempts, and automated scoring with attempt history.
