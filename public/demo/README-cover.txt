Profile cover image
------------------
To use your own cover image (e.g. "Life Is A Journey"):

1. Save or copy your image file here as:  profile-cover.png
2. The profile page will show it automatically as the cover banner.

Supported: PNG, JPG, or WebP. Recommended size: about 1200×400 px (wide).

If profile-cover.png is missing, the app falls back to the default SVG cover.
